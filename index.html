Shortcut syntax


						$ app/console d:s:v

The above is equivalent to:


						$ app/console doctrine:schema:validate

If the shortcut is ambiguous


						$ app/console d:g:e



  [InvalidArgumentException]
  Command "d:g:e" is ambiguous (doctrine:generate:entities, doctrine:generate:entity).
